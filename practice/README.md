# Practice — scenario drills & mock exam

> 🎯 **Prefer instant feedback?** All of these questions are also available as an **interactive quiz** —
> [`quiz.html`](https://thedocwho.github.io/claude-architect-certification/quiz.html) — with per-answer explanations, score tracking, and a mock-exam mode. These
> Markdown files are the same content in printable/reading form.

The real exam is **scenario-based multiple choice**: a situation is described, and the wrong answers *sound* like
good engineering. These drills mirror that style. For each question:

1. Read the scenario's **constraints first** — they decide the answer.
2. Pick before peeking. Then open the answer and read **why the distractors are wrong**, not just why the key is
   right. The "why wrong" is where the learning is.

> These are **study aids written for this track**, not real exam questions. They drill the same judgment.

## How to review a miss
Every wrong answer maps to one of the [14 exam traps](../BLUEPRINT.md#high-frequency-exam-traps-the-distractors-that-look-smart-but-are-wrong).
When you miss one, find which trap fooled you and re-drive the matching [explainer](https://thedocwho.github.io/claude-architect-certification/illustrated/index.html).
Misses are data — they tell you exactly which intuition to rebuild.
